# Sales Performance Analysis

## Overview
Analyzed structured sales data to extract business insights using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Electronics is the highest revenue-generating category
- Home & Furniture has the highest per-order profitability
- Accessories require pricing improvements to increase margins

## Dataset
Product sales dataset with revenue, profit, category and region data

## Files
- Sales_Performance_Analysis.ipynb — Main analysis notebook
